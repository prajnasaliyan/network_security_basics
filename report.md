## Network Threats
- Virus: Malicious code that infects files
- Worm: Self-spreading malware over networks
- Trojan: Disguised malicious software
- Phishing: Social engineering attack to steal information

## Security Measures Implemented
- Enabled UFW firewall
- Used NAT network for isolation
- Applied strong user authentication
- Monitored encrypted HTTPS traffic

## Traffic Analysis
Wireshark was used to capture live network traffic.
DNS, TCP, and TLS packets were identified during normal browsing.

## Conclusion
Basic security configurations such as firewalls and traffic monitoring significantly improve
network protection and awareness.
