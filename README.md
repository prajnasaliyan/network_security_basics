# Network Security Basics using Kali Linux

## Description
This project demonstrates basic network security concepts using a Kali Linux virtual machin>
It focuses on identifying common network threats, implementing firewall protection, and
monitoring network traffic using Wireshark.

## Tools Used
- Kali Linux (Virtual Machine)
- UFW Firewall
- Wireshark
- VirtualBox / VMware (NAT Network)

## Key Concepts Covered
- Network threats (Virus, Worm, Trojan, Phishing)
- Firewall configuration (UFW)
- Network traffic analysis
- Secure network setup using NAT

## Practical Work
- Enabled and configured firewall in Kali Linux
- Captured and analyzed DNS, TCP, and HTTPS traffic
- Observed encrypted communication and normal network behavior

## Outcome
This project provides a foundational understanding of how basic security measures help prot>
a network from common threats.
